# COMPSCI 383 - F24 Assignments

This archive contains the materials used in COMPSCI 383 Artificial Intelligence taught at University of Massachusetts Amherst. Each folder corresponds to an individual assignment and may include prompts, rubrics, example code, or supplementary materials.
